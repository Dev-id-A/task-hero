# This is a personal project about a task app
It’s a project where I decided to try out TypeScript and add a small testing setup to ensure the logic works correctly. I also used Tailwind together with Framer Motion for styling and animations.

The app is built around a set of accordions that display different types of tasks. Each task has its own difficulty level, and depending on that, you earn a certain amount of experience. As you level up, you unlock a series of fun “errand runner” titles.

---
# Esto es un proyecto personal sobre una aplicación de tareas
Es un proyecto donde decidí probar TypeScript y añadir una capa de testing para asegurar la lógica. También utilicé Tailwind junto a Framer Motion para los estilos y las animaciones.

La aplicación funciona con un sistema de acordeones que organiza distintos tipos de tareas. Cada tarea tiene un nivel de dificultad, y según cuál sea, ganas una cantidad diferente de experiencia. Al subir de nivel, desbloqueas varios motes divertidos relacionados con ser “recadero”.
